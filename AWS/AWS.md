# AWS
```
AWS는 아마존닷컴의 클라우드 컴퓨팅 사업부(Amazon Web Services)를 뜻함
물리적인 서버를 직접 구축할 필요 없이, 인터넷을 통해 필요한 만큼의 IT 인프라(서버, 데이터베이스, 스토리지 등)를 빌려 쓰고 사용한 만큼만 요금을 지불하는 대표적인 클라우드 서비스
```

## Region과 Availability Zone

### Region
```
AWS 데이터 센터가 존재하는 하나의 지리적 지역
```
- 리전 선택 요소
    - 사용자와의 거리 및 지연 시간
    - 서비스 제공 여부
    - 데이터 보관 관련 법률
    - 비용
    - 재해 복구 전략

### Availability Zone
```
AZ는 하나의 Region 안에 존재하는 서로 독립된 물리적 위치
```
- 각 AZ는 하나 이상의 데이터센터로 구성 
- 전력, 네트워크, 연결 설비가 이중화되어 있다. 
- 같은 Region의 AZ들은 빠른 전용 네트워크로 연결되지만, 하나의 장애가 다른 AZ로 번지는 것을 줄이도록 물리적으로 분리
- 서버를 서로 다른 AZ에 배치해야 장애 범위를 분리할 수 있다.
- Multi-AZ
    - 여러 AZ에 자원을 배치하여 하나의 AZ 장애에 대비하는 구성
    - 하나의 AZ의 장애가 생기더라도 다른 AZ의 서버가 동작
    - 이것을 고가용성, High Availability이라고 한다.
